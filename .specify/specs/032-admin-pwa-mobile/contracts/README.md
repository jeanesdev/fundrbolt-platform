# API Contracts: Admin PWA Mobile & Tablet UI

**Feature**: 032-admin-pwa-mobile

## No API Contracts

This feature is entirely frontend. No new API endpoints are introduced, and no existing endpoints are modified.

All data flows use the existing API endpoints already consumed by the admin PWA. The only new "contract" is the client-side localStorage schema documented in [data-model.md](../data-model.md).
